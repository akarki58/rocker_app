Rocker Reporting Application

Collect data to single Excel from various data sources, Use Excel templates to present business graphs. Combine data with Excel Power Pivot. Get data from Odoo & external PostgreSQL, SQLServer, MySQL, MariaDB, ODBC or Oracle databases.

Requirements:

  1. Odoo 12 installed on Windows platform
  2. Excel 32-bit installed on server
  3. Win32Com installed on server
  4. Datasource drivers (PostgreSQL, SQLServer, Oracle, MySQL, MariaDB etc.) installed on server based on your needs

Examples:

  1. Rocker example: Data from single datasource


  2. Rocker example: Collection reports, data from many datasources
