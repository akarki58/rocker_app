Rocker Reporting Application

for Odoo 17 / Odoo16 / Odoo 15 / Odoo 12.0 / Odoo 13.0 / Odoo 14.0

Collect data to single Excel from various data sources, Use Excel templates to present business graphs. Combine data with Excel Power Pivot. Get data from Odoo & external PostgreSQL, SQLAlchemy, SQLServer, MySQL, MariaDB, ODBC or Oracle databases.
